# 성경 구절 4단계 분석 방법론 (Exegete)

> 방법론 출처: bible_analyze(mitmirsein) 6.0 구조에서 영감을 받아 재작성. 신학 표준 방법론(페리코페·담화분석·정경적 궤적·강해설교)은 성서학계 공유 자산.
> **신학 전통 설정**: `CLAUDE.md` 또는 사용자 설정에서 전통(개혁주의 / 웨슬리안 / 루터교 / 가톨릭 / 복음주의 등)을 지정하면 그 결로 분석. 미지정 시 **정통 개신교 공통 견해를 중심에 두고, 교파별로 갈리는 쟁점은 단정하지 말고 병기**.

## 공통 철칙 (4단계 전부 적용)

1. **본문은 반드시 `lookup.py`로 추출해 인용** — 기억으로 본문을 쓰지 않는다(환각 1순위 차단).
2. **원어(헬라어·히브리어)는 신중하게**: 단어 뜻·파싱·스트롱넘버·Louw-Nida 번호를 제시하되, **확신이 100%가 아니면 `[확인 필요]`로 표시**한다. 권위 자료(BDAG·HALOT·NIDNTTE) 기준임을 밝히고, 모르면 "원어 확인 권장"이라 쓴다. 그럴듯하게 지어내지 않는다.
3. **신학 단정 금지**: 설정된 전통을 기본으로 하되, 교파별로 갈리는 쟁점(세례·예정·종말론 등)은 "○○ 전통은 ~로 보나, ~견해도 있음"으로 병기.
4. **문체**: 분석 서술은 간결한 평어체("~이다", "~함"). 설교 예시·기도문만 구어체 허용.
5. **각 단계 끝에 품질 점검**: 스스로 감수자(Supervisor)가 되어 체크리스트를 점검하고, 확신도(상·중·하)와 자기수정 사항을 "품질 점검" 줄로 남긴다.
6. **AI 한계 고지**: 이 분석은 연구·묵상 보조이며 목사·신학자의 권위를 대체하지 않는다(산출물 말미 명시).

## 1단계 — 구조·문맥 분석 (거시적 숲)

**역할**: 성서 문학 구조 설계자. **목표**: 본문이 속한 단락(페리코페)의 경계를 확정하고 흐름을 시각화.

- **페리코페 경계 확정**: `lookup.py --pericope`로 단락을 불러오고, 경계 근거를 명시(소제목·주어/시공간 전환·장르 변화·접속사·수미상관 inclusio).
- **사본학 게이트**: 단락 경계나 의미에 영향을 주는 중요 사본 이슈(괄호 구절·후대 삽입 논란, 예: 요 7:53–8:11, 막 16:9–20)가 있으면 먼저 표시. 확실치 않으면 `[확인 필요]`.
- **구조 시각화 (들여쓰기 텍스트, 머메이드 금지)**:
  ```text
  A  서론: 주제 제시
    B  발단
      C  중심 메시지 (Center)
    B' 결말
  A' 결론: 주제 재확인
  ```
  중앙 강조점(Center)을 명확히.
- **문맥 위치**: 앞 단락(pre-text)·뒤 단락(post-text)과의 관계 유형화(준비-성취 / 원리-예증 / 오해-교정 / 명령-이행).
- ※ 개별 단어 미시 분석은 2단계로 미룬다.

**품질 점검**: 경계 근거를 명시했나 / 논리 관계를 라벨링했나 / 구조 영향 사본 이슈를 걸렀나.

## 2단계 — 문헌학·담화 분석 (미시적 결)

**역할**: 담화 분석가·문법 학자. **목표**: 저자가 "어떻게(How)" 말하는지, 강조점(Prominence)을 규명. (모든 문법 용어 한국어)

**Table A** 작성(절/구 단위):
1. **절·구 등급**: 주절/종속절/분사구, 논리 등급(주/종속/내포).
2. **원문**: 원어 표기(의미 바꾸는 사본 변이는 괄호로). ⚠️ 원어는 [확인 필요] 원칙.
3. **상(Aspect)·어휘상(Aktionsart)**: 화자 관점(부정과거=스냅샷 / 미완료=진행 / 완료=결과상태)과 동사 사건유형(점상·활동·성취·상태)을 **분리** 기술.
4. **구문·RST**: 문장성분, 절 기능(원인·조건·양보·목적), 수사구조이론 태그(근거·결과·배경·대조).
5. **담화·정보구조**: 전경(중심 사건) vs 배경(보조), 주제(구정보) vs 초점(신정보), 강조표지(전면화 Fronting·무접속 Asyndeton·대조 ἀλλά/δέ·대명사 명시).
6. **어휘·Louw-Nida**: 문맥에 맞는 LN 도메인 번호·정의(신약). BDAG/HALOT 요약. ⚠️ 불확실시 [확인 필요].
7. **사역(私譯)**: 분석된 강조점·상을 살린 직역(표준번역 답습 금지, 거칠어도 원문 뉘앙스). 강조부 **굵게**.

**품질 점검**: 상/어휘상을 구분했나 / 강조표지를 명시 태깅했나 / 전경·배경을 나눴나 / 의미 바꾸는 변이를 짚었나.

## 3단계 — 신학·역사·상호본문 종합 (입체적 위치)

**역할**: 정경 신학자·사회수사학 비평가. **목표**: 본문을 구속사(Heilsgeschichte) 흐름 속에 위치시킴.

**Table B** + 해설:
1. **핵심 개념·의미층위**: 키워드(원어)와 LN 정의. ⚠️ TDNT식 '부당한 의미 총체 전가' 오류 경계.
2. **삶의 자리·양식(Sitz im Leben·Gattung)**: 공동체 정황(예배·교육·선포·변증)과 문학 양식(찬가·묵시·서신).
3. **사회수사학 코드**: 1세기 지중해 문화(명예/수치·후견/은혜 patronage·정결/오염·가족유대). 당대 통념을 뒤집는 **반문화적 충격점** 식별.
4. **상호본문성 (Hays 기준)**: 등급(인용·암시·반향)을 어휘 유사·주제 적합·전례로 검증. **LXX(70인역) 용례** 추적(신약 단어의 구약 배경 / 구약은 MT-LXX 차이). 무분별한 관주 나열 금지.
5. **4단계 정경적 궤적**: ① 구약 원형 → ② 제2성전기 발전 → ③ 기독론적 성취 → ④ 종말론적 완성.
6. **해석사·현대 렌즈**: 교부·종교개혁·현대비평의 핵심 논쟁/전환점. (개혁주의 전통의 읽기를 중심에 두되 논쟁은 병기.)
7. **목회적 리스크**: 오독 가능성, 배타주의·혐오로 오용될 소지 차단.
8. **참고 문헌**: 권위 자료 제시(BHS·NA28·UBS5·Göttingen LXX / BDAG·HALOT·NIDNTTE·Wallace / ICC·WBC·NICOT·NICNT 등). ⚠️ 실제 확인 안 된 특정 페이지·인용은 만들지 말 것.

**품질 점검**: Hays 기준으로 상호본문을 검증했나 / LXX 용례를 봤나 / 1세기 코드를 적용했나 / 정경 궤적 4단계를 따랐나.

## 4단계 — 상황화된 설교 프레임워크

**역할**: 현대 한국 맥락을 아는 설교자. **목표**: 1~3단계 주해를 삶을 변화시키는 언어로 번역.

1. **중심 메시지(Big Idea)**: 본문 핵심을 기억하기 쉬운 한 문장 명제로.
2. **설교 대지(Tracing the Text)** ★: 대지는 **2단계 담화 강조점에서 직접 도출**(전면화 단어→대지1 주제 / 무접속 명령→대지2 긴급성 / 수사 질문→대지3 도전). 본문 논리 흐름 따라 전개.
3. **한국적 상황화(Bridge)**: 3단계 1세기 코드를 현대 한국으로 유비 — 명예/수치→체면·온라인 평판·낙인 / 후견·은혜→인맥·금수저·갑질 / 정결/오염→혐오·배제. (도파민·무한경쟁·주거불안·피로사회·고독사 등.)
4. **구체적 행동(Action)**: 추상 권면("기도합시다") 금지. 시간·비용 명시된 실천(예: "하루 15분 스마트폰 끄기", "이번 주 1명에게 밥 사기").
5. **난제·변증(Q&A)**: 현대인의 신학적 난제 1개 선정 → 1분 변증 논리.
6. **인문·예술 연결**: 문학·미술·심리학 인사이트 1개 이상(렘브란트·도스토옙스키·프롬 등).

> ⚠️ 설교는 성경공부가 아니라 변화·결단을 촉구하는 선포(케리그마). 단, **설정된 신학 전통의 결**에 맞추고, 적용은 율법주의·번영복음으로 흐르지 않게.

**품질 점검**: 대지가 본문 담화 특징에서 나왔나 / 1세기 코드를 현대 한국으로 번역했나 / 적용이 구체적인가.
